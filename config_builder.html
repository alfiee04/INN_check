Конструктор config.json

Собирает файл конфигурации для скрипта проверки контрагентов — без сервера, всё в браузере.

1. Загрузить существующий config.json (необязательно)

Если файл уже есть — загрузите его, чтобы отредактировать и скачать заново.

Перетащите сюда config.json или нажмите, чтобы выбрать файл

2. Общие настройки

Применяются ко всем проверкам в этом конфиге.

3. Массовая загрузка списка ИНН

Файл .txt или .csv. Один контрагент — одна строка. Поддерживаются форматы:
• только ИНН: 1616019804
• ИНН и данные через запятую: 1616019804, ООО «Ромашка», Иванов И.И., г. Казань, ул. Ленина, 1
(порядок колонок: ИНН, название компании, исполнитель, адрес)

Перетащите сюда .txt/.csv со списком ИНН или нажмите, чтобы выбрать файл
…или вставьте список вручную

4. Компании для проверки 0

Добавляйте по одной вручную или редактируйте загруженный список прямо в таблице.
